## REMIX TECHNIQUES

Once you find a direction you like, remix it by:

**Adding words to emphasize specific qualities:**
- "bold linework" = thicker, stronger lines
- "delicate precision" = thinner, more refined lines
- "technical drawing" = engineering/blueprint feel
- "modern emblem" = more iconic/badge-like
- "geometric precision" = perfect angles/measurements

**Adjusting element counts:**
- "2 circuit traces" = minimal
- "3 circuit traces" = balanced (recommended)
- "5 circuit traces" = network feeling
- "3-4 connection nodes" = flexibility

**Changing node style:**
- "circular nodes" = simple dots
- "connection nodes" = more complex
- "geometric nodes" = square/angular
- "PCB solder points" = authentic circuit board feel

## COMMON ISSUES & FIXES

**Issue:** Anvil shape not recognizable
**Fix:** Add "with horn and base", "classic anvil shape", or "blacksmith anvil"

**Issue:** Circuit traces too complex/messy
**Fix:** Add "clean straight lines", "minimal branching", "simplified pathways"

**Issue:** Proportions off (anvil too small, traces too big)
**Fix:** Specify "large anvil base with small circuit traces" or vice versa

**Issue:** Too many elements/busy
**Fix:** Add "ultra minimal", "2-3 elements maximum", "simplified"

**Issue:** Not geometric enough
**Fix:** Add "geometric precision", "isometric technical drawing", "CAD style"

**Issue:** Colors not working
**Fix:** Generate in black/white first, add color in post-processing for precision

## POST-MIDJOURNEY WORKFLOW

### 1. Select Winner
- Choose the concept that best captures the brand
- Ideally select 1-2 finalists for comparison

### 2. Upscale & Export
- Upscale to maximum resolution in Midjourney
- Download PNG files

### 3. Vector Conversion
- Import to Adobe Illustrator or Figma
- Use Image Trace (Illustrator) or manual tracing
- Clean up paths for perfect geometry

### 4. Refine Geometry
- Adjust angles to perfect 30¬∞ isometric
- Align nodes to pixel grid
- Perfect curve smoothness
- Ensure consistent line weights

### 5. Apply Brand Colors
- Forge Black: #1A1A1A
- Spark Orange: #FF6B35 (for accent nodes)
- Vector Blue: #4A90E2 (alternative)

### 6. Create Variations
- Icon only (no wordmark)
- Horizontal lockup (icon + ARCHFORGE)
- Stacked lockup (icon over wordmark)
- Monochrome versions (all black, all white)
- Favicon (simplified for 16x16px)

### 7. Export Production Files
- SVG (primary scalable format)
- PNG: 512px, 256px, 128px, 64px, 32px, 16px
- PDF (for print applications)
- Favicon.ico (for web)
